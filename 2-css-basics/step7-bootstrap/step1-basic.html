📌 문단과 텍스트 스타일

📖 Lead 문단: 중요한 소개글이나 요약에 사용하는 큰 문단입니다. 일반 문단보다 크고 굵어서 시선을 집중시킵니다.

일반적인 문단입니다. 부트스트랩은 기본적으로 가독성이 좋은 폰트와 적절한 줄 간격을 제공합니다. 하이라이트된 텍스트나 작은 부가 설명도 쉽게 만들 수 있습니다.

인용문은 이렇게 표현됩니다.

4 텍스트 색상과 정렬 유틸리티

📌 의미적 텍스트 색상

각 색상이 전달하는 의미를 활용

Primary: 주요 브랜드 색상

Secondary: 보조 색상

Success: 성공, 완료, 긍정적 메시지

Info: 정보 제공, 안내

Warning: 주의사항, 경고

Danger: 오류, 위험, 삭제

Light: 밝은 텍스트 (어두운 배경에서)

Dark: 어두운 텍스트

Muted: 부가 설명이나 덜 중요한 정보

📌 텍스트 정렬

  • CDN 한 줄로 즉시 변화: 부트스트랩 CSS 파일 하나만 연결해도 놀라운 변화
