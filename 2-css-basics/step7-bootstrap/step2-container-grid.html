🚨 자주 하는 실수와 해결방법

실수를 통해 더 깊이 이해하기

❌ 실수 1: row 없이 col 사용

잘못된 예시:

<div class="container">
  <div class="col-6">내용</div>
</div>

올바른 예시:

<div class="container">
  <div class="row">
    <div class="col-6">내용</div>
  </div>
</div>

❌ 실수 2: 12를 넘는 컬럼 배치

col-8 + col-6 = 14 (12를 넘음)

col-8
col-6
(다음 줄로 이동됨)

올바른 해결방법:

col-6 + col-6 = 12 또는 새로운 row 생성

🎯 실무 활용: 전형적인 웹사이트 레이아웃

실제 프로젝트에서 사용하는 구조

🏠 헤더 영역 (Header)

로고 + 네비게이션 + 사용자 메뉴

📝 메인 콘텐츠 (Main Content)

핵심 정보, 기사 내용, 상품 설명

전체의 66.7% 공간 차지
📋 사이드바 (Sidebar)

광고, 관련 링크, 부가 정보

전체의 33.3% 공간 차지
👣 푸터 영역 (Footer)

회사 정보 + 저작권 + 사이트맵

🎯 오늘의 핵심 정리

📐 그리드 시스템 3단계
  1. Container: 전체 영역 정의
  2. Row: 가로 행 생성
  3. Column: 세로 열 배치
🔢 12컬럼 시스템
  • 한 행은 항상 12개 컬럼
  • 균등분할: col (자동)
  • 비율지정: col-4, col-8 등
