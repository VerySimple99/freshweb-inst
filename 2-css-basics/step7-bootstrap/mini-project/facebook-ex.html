📋 프로젝트 개요

🎯 프로젝트 목표
  • ✅ 반응형 네비게이션 구현
  • ✅ Bootstrap 그리드 시스템 활용
  • ✅ 카드 컴포넌트로 정보 구성
  • ✅ 모바일/태블릿/데스크톱 최적화
  • ✅ 실무에서 사용 가능한 완성도
📱 완성 후 결과물
  • 🏠 네비게이션: 이름 + 메뉴 + 모바일 토글
  • 🎪 히어로 섹션: 소개 + CTA 버튼
  • 👤 소개 카드: 프로필 + 스킬 + 경력
  • 📞 연락처 카드: 이메일 + SNS 링크
  • 📱 완전 반응형: 모든 기기 최적화

🚀 단계별 구현 가이드

1단계: HTML 기본 구조 만들기 (5분)

🎯 목표: Bootstrap CDN 연결 및 기본 HTML 구조 완성

💡 핵심 포인트: DOCTYPE → head 설정 → body 구조 순서로 차근차근
📝 코드 템플릿:
<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>홍길동 - 웹 개발자</title> <!-- 🎯 Bootstrap 4 CDN --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/css/bootstrap.min.css"> <!-- 🎨 Font Awesome (아이콘용) --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css"> </head> <body> <!-- 여기에 내용 작성 --> <!-- 🔧 Bootstrap JavaScript --> <script src="https://cdn.jsdelivr.net/npm/jquery@3.5.1/dist/jquery.slim.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
✅ 체크포인트: 파일을 저장하고 브라우저에서 열어보세요. 빈 페이지가 정상적으로 표시되면 성공!

2단계: 네비게이션 바 만들기 (10분)

🎯 목표: 이름(브랜드) + 메뉴 + 모바일 햄버거 메뉴 완성

💡 핵심 포인트: .navbar → .container → .navbar-brand + .navbar-toggler + .navbar-collapse
📝 코드 템플릿:
<!-- 🧭 네비게이션 바 --> <nav class="navbar navbar-expand-lg navbar-light bg-white shadow-sm"> <div class="container"> <!-- 📛 브랜드/이름 --> <a class="navbar-brand font-weight-bold text-primary" href="#"> 홍길동 </a> <!-- 🍔 모바일 토글 버튼 --> <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav"> <span class="navbar-toggler-icon"></span> </button> <!-- 📱 반응형 메뉴 --> <div class="collapse navbar-collapse" id="navbarNav"> <ul class="navbar-nav ml-auto"> <li class="nav-item active"> <a class="nav-link" href="#home">홈</a> </li> <li class="nav-item"> <a class="nav-link" href="#about">소개</a> </li> <li class="nav-item"> <a class="nav-link" href="#contact">연락처</a> </li> <li class="nav-item"> <a class="nav-link btn btn-outline-primary ml-2 px-3" href="#">이력서 다운로드</a> </li> </ul> </div> </div> </nav>
🔍 주의사항:
  • data-target="#navbarNav"와 id="navbarNav"가 일치해야 함
  • 브라우저 창 크기를 조절하며 모바일 메뉴 동작 확인
  • 본인 이름으로 "홍길동" 부분 변경하기
✅ 체크포인트: 브라우저 창을 좁혀서 햄버거 메뉴가 나타나는지 확인하고, 클릭했을 때 메뉴가 펼쳐지는지 테스트해보세요.

3단계: 히어로 섹션 만들기 (10분)

🎯 목표: 임팩트 있는 자기소개 + 행동 유도 버튼

💡 핵심 포인트: .container → .row → .col → 중앙 정렬 + 큰 텍스트 + 버튼
📝 코드 템플릿:
<!-- 🎪 히어로 섹션 --> <section id="home" class="py-5" style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);"> <div class="container"> <div class="row min-vh-50 align-items-center"> <div class="col-12 text-center text-white"> <h1 class="display-3 font-weight-bold mb-3">안녕하세요! 👋</h1> <h2 class="h1 mb-4">저는 <span class="text-warning">홍길동</span>입니다</h2> <p class="lead mb-4"> 사용자 중심의 웹 서비스를 만드는 <strong>프론트엔드 개발자</strong>입니다.<br> React와 Vue.js로 혁신적인 사용자 경험을 구현합니다. </p> <div class="mt-4"> <a href="#about" class="btn btn-warning btn-lg mr-3"> <i class="fas fa-user mr-2"></i>더 알아보기 </a> <a href="#contact" class="btn btn-outline-light btn-lg"> <i class="fas fa-envelope mr-2"></i>연락하기 </a> </div> </div> </div> </div> </section>
🎨 커스터마이징 팁:
  • 이름과 직업: 본인 정보로 변경
  • 소개글: 본인의 강점이나 관심 분야로 수정
  • 색상: text-warning를 text-info, text-success 등으로 변경 가능
  • 배경: 그라데이션 색상을 본인 취향으로 조정
✅ 체크포인트: 히어로 섹션이 화면 가득 채워지고, 버튼이 제대로 보이는지 확인해보세요. 모바일에서도 텍스트가 읽기 쉬운지 체크!

4단계: 소개 카드 섹션 만들기 (15분)

🎯 목표: 프로필 사진 + 상세 소개 + 스킬 정보를 카드로 구성

💡 핵심 포인트: .container → .row → .col-md-4 + .col-md-8 (1:2 비율)
📝 코드 템플릿:

🖼️ 프로필 사진 교체 방법:
  1. 본인 사진을 "profile.jpg" 파일명으로 저장
  2. HTML 파일과 같은 폴더에 업로드
  3. src="https://via.placeholder.com/..." 부분을 src="profile.jpg"로 변경
✅ 체크포인트: 프로필 카드와 소개 카드가 데스크톱에서는 나란히, 모바일에서는 세로로 배치되는지 확인해보세요.
