안녕하세요! 사용자 경험을 최우선으로 생각하는 프론트엔드 개발자 홍길동입니다.

3년간 다양한 웹 프로젝트를 통해 React, Vue.js, JavaScript ES6+를 활용한 모던 웹 애플리케이션 개발 경험을 쌓았습니다. 특히 사용자가 직관적으로 사용할 수 있는 인터페이스 설계와 성능 최적화에 관심이 많습니다.

새로운 기술을 배우는 것을 즐기며, 팀과의 협업을 통해 더 나은 결과물을 만들어내는 것에 보람을 느낍니다. 코드의 가독성과 유지보수성을 중시하며, 항상 사용자 관점에서 생각하려 노력합니다.
