🧭 네비게이션 핵심 포인트
  • 구조: .navbar → .navbar-brand + .navbar-nav
  • 반응형: .navbar-expand-{breakpoint}로 제어
  • 토글: 모바일 햄버거 메뉴 자동 생성
  • 스타일: .navbar-light/dark + bg-* 조합
🃏 카드 핵심 포인트
  • 구조: .card → .card-body가 기본
  • 확장: header, footer, 이미지 추가 가능
  • 일관성: 같은 용도는 동일 구조 유지
  • 반응형: 그리드와 조합으로 배치 제어
🎨 디자인 원칙 요약

팀 카드 .card .text-center 신뢰감, 전문성 표현 프로필 사진, 역할, 소개
